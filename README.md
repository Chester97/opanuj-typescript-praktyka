# Opanuj TypeScript - Repozytorium dla uczestnika

## Wymagania

- Node.js 20

## Instalacja

```bash
npm install
```

## Uruchomienie zadania z modułu

- npm run list <moduł>

Przykład:

```bash
npm run list core-pro
```

## Uruchomienie zadania wg nazwy

- npm run verify <zadanie>

Przykład:

```bash
npm run verify type-filtering
```
